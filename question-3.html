¿Que es el lenguaje de programación Java?

Java es un lenguaje de programación orientado a objetos rápido, seguro y confiable para codificarlo todo desde aplicaciones móviles y software empresarial hasta aplicaciones de macrodatos y tecnologías del servidor. Fue diseñado inicialmente por Sun Microsystems (que ahora es parte de Oracle). Uno de sus principales distintivos es la capacidad de poder escribir una vez y ejecutar en cualquier lugar, lo que significa que el código Java compilado puede ejecutarse en cualquier sistema que tenga una JVM (Maquina Virtual de Java).

Ventajas

•Portabilidad y su JVM (Maquina virtual de Java)

•Seguridad y fiabilidad

•Comunidad y Soporte

•Rendimiento optimizado y escalabilidad

•Ecosistema rico en herramientas de desarrollo

Desventajas

•Rendimiento y uso de recursos

•Curva de aprendizaje

•Dependencia de la JVM (Maquina virtual de Java)

•Interfaz gráfica de usuario (GUI)
